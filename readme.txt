# Savory Haven

**Savory Haven** is a restaurant website designed to showcase our delicious menu and provide customers with essential information about our establishment. The website offers an easy-to-navigate interface where users can explore the menu, learn about our history, and make reservations.

## Features

- **Responsive Design**: The website is fully responsive, ensuring a seamless experience across all devices.
- **Menu Sections**: Users can browse through different meal options, including breakfast, lunch, dinner, and desserts.
- **Reservation Form**: A simple form is provided for customers to reserve a table.
- **Newsletter Subscription**: Visitors can subscribe to our newsletter to receive updates and promotions.
- **Google Maps Integration**: The site includes an embedded Google Map for easy location access.

## Technologies Used

- **HTML5**: The structure of the website.
- **Bootstrap 5**: For responsive design and components.
- **Font Awesome**: Icons used throughout the site.
- **Google Fonts**: Custom fonts for improved aesthetics.
- **JavaScript (jQuery)**: For interactive elements and form handling.
